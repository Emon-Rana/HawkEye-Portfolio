# HawkEye-Portfolio
It's a simple personal portfolio website developed by using html,css,bootstrap,javascript,php and mysql

## Environment
* xampp as local server
* editor like notpad++ / vs
## Installation
* Download the zip file of the code
* Extract the zip file
* Copy the folder in xampp > htdocs
* Import the sql file in  phpmyadmin

That's it, now you can run the project

## Feature
* User validation 
* edit portfolio
